## fbx2gltf.py 使用方法

### 安裝 Fbx python 

https://www.autodesk.com/products/fbx/overview

預設安裝位置
C:\Program Files\Autodesk\FBX\FBX Python SDK\2018.1.1\lib

### 安裝 python

一定要安裝跟 FBX sdk 對應的版本

比如 Python fbx 2018.1.1 提供了 2.7 跟 3.3 的函式庫
那就你的 Python 就一定要是 2.7.x 或 3.3.x

不要安裝 3.5 or 3.7，不相容
